## Response format

For every PR, AI responds using this structure

1. PR
   - name
   - scope in 1–2 sentences

2. Summary
   - 3–6 bullets of what changed

3. Files
   - New
   - Updated

4. Changes
   - New files
     - purpose
     - key sections
   - Existing files - key changes - reasoning

5. Tests
   - commands to run
   - what they cover

6. Notes
   - risks, tradeoffs, follow-ups

7. Docs
   - docs changed or none
